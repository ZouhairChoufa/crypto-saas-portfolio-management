# Screenshots

Ce dossier contient les captures d'écran du projet CryptoSaaS.

## Structure recommandée

```
screenshots/
├── dashboard-market.png          # Page principale avec graphique dynamique
├── dashboard-news.png            # Dashboard d'analyse des actualités
├── dashboard-twitter.png         # Dashboard de sentiment Twitter
├── dashboard-telegram.png        # Dashboard des signaux Telegram
├── dark-mode.png                 # Interface en mode sombre
├── light-mode.png                # Interface en mode clair
├── mobile-responsive.png         # Version mobile
└── chart-interaction.png         # Graphique avec tooltips
```

## Instructions pour les captures

### Résolution recommandée
- **Desktop:** 1920x1080 ou 1440x900
- **Mobile:** 375x812 (iPhone X) ou 360x640 (Android)

### Contenu à capturer
1. **Dashboard Market** - Graphique avec données réelles Bitcoin
2. **Dashboard News** - Liste d'articles avec sentiment
3. **Dashboard Twitter** - Hype meter et feed de tweets
4. **Dashboard Telegram** - Signaux de trading
5. **Dark/Light Mode** - Comparaison des deux thèmes
6. **Mobile** - Version responsive sur smartphone

### Outils recommandés
- **Windows:** Outil Capture d'écran (Win + Shift + S)
- **Mac:** Capture d'écran (Cmd + Shift + 4)
- **Extensions:** Full Page Screen Capture (Chrome)

### Format
- **Format:** PNG (meilleure qualité)
- **Compression:** Optimisée pour web
- **Taille max:** 2MB par image
